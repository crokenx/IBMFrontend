## Descripcion reactive forms
Reactive forms de angular, provee distintos metodos por los cuales validar el estado de los campos que componen un formulario, volviendolo estricto, es decir, recibiendo solo el tipo de data requerida por el formulario, evitando de esta manera el envio de datos erroneos a las API's que correspondan.
